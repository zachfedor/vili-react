# vili-client

The simplest and most intuitive time tracker ever.


## Installation

* `git clone <repository-url>`
* `cd vili-react`
* `npm install`


## Development

* `npm start`
* Visit your app at [http://localhost:8080](http://localhost:8080).


## Other Scripts

* `npm test`
* `npm run test:watch`
* `npm run lint`
* `npm run build`
